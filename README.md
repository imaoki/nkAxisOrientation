## 開発環境

`Maya 2022.5`

## インストール

01. ファイルをコピーする。

    英語版
    : `%MAYA_APP_DIR%\<version>\scripts`

    日本語版
    : `%MAYA_APP_DIR%\<version>\ja_JP\scripts`

    複数バージョン
    : `%MAYA_APP_DIR%\scripts`

02. Mayaを再起動

## 使い方

```mel
nkAxisOrientation;
```
